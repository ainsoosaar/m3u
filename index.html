О ПРОЕКТЕ

POKAZ-M3U - полностью автоматизированная система для сбора актуальных M3U плейлистов с сайта pokaz.me и их публикации на GitHub. Система работает полностью автономно, запускаясь каждые 6 часов через планировщик задач Windows.

Текущий статус: Система полностью работоспособна и автоматизирована

Веб-интерфейс для мониторинга

production.ps1 - Главный скрипт автоматизации v10.0.0

# ============================================
# production.ps1 - ЗАПУСК КАЖДЫЕ 6 ЧАСОВ
# ============================================

$playlistsPath = "C:\Users\megaa\Desktop\Raschirenija\pokaz-m3u-clean\playlists"
$playlistFile = "pokaz_playlist.m3u8"
$githubToken = [Environment]::GetEnvironmentVariable("GITHUB_TOKEN", "User")
$repoOwner = "ainsoosaar"
$repoName = "m3u"
$branch = "main"

# Создаём имя файла с текущей датой и временем
$githubFileName = "playlist_from_pc_$(Get-Date -Format 'yyyyMMdd_HHmmss').m3u8"
$fullPlaylistPath = Join-Path $playlistsPath $playlistFile

Write-Host "🚀 ЗАПУСК ПРОЦЕССА (6h schedule)" -ForegroundColor Cyan

# ШАГ 1: Проверка токена
if (-not $githubToken) {
    Write-Host "❌ Токен не найден!" -ForegroundColor Red
    exit 1
}

# ШАГ 2: Сборка плейлиста
Push-Location $playlistsPath
node build_pokaz_fixed.js
Pop-Location

# ШАГ 3: Проверка файла
if (-not (Test-Path $fullPlaylistPath)) {
    Write-Host "❌ Файл не найден!" -ForegroundColor Red
    exit 1
}

# ШАГ 4: Кодирование в Base64
$bytes = [System.IO.File]::ReadAllBytes($fullPlaylistPath)
$base64 = [Convert]::ToBase64String($bytes)
$base64Escaped = $base64 -replace '"', '\"' -replace '\\', '\\\\'

# ШАГ 5: Отправка на GitHub
$apiUrl = "https://api.github.com/repos/$repoOwner/$repoName/contents/$githubFileName"
$jsonString = '{"message":"Auto-update 6h schedule","content":"' + $base64Escaped + '","branch":"' + $branch + '"}'

$tempFile = Join-Path $env:TEMP "github_prod.json"
$jsonString | Out-File $tempFile -Encoding ASCII -NoNewline

$response = curl.exe -s -X PUT `
    -H "Authorization: token $githubToken" `
    -H "Accept: application/vnd.github+json" `
    -H "Content-Type: application/json" `
    -d "@$tempFile" `
    "$apiUrl"

$result = $response | ConvertFrom-Json

if ($result.content) {
    Write-Host "✅ УСПЕХ! Файл создан: $githubFileName" -ForegroundColor Green
    Write-Host "🔗 https://github.com/$repoOwner/$repoName/blob/$branch/$githubFileName"
} else {
    Write-Host "❌ Ошибка: $($result.message)" -ForegroundColor Red
}

Remove-Item $tempFile -Force -ErrorAction SilentlyContinue

Проблема Вероятная причина Решение
❌ Токен не найден Переменная окружения не установлена [Environment]::SetEnvironmentVariable("GITHUB_TOKEN", "ваш_токен", "User")

❌ Планировщик не запускается Задача отключена или нет прав Enable-ScheduledTask -TaskName "Pokaz-M3U Production"
