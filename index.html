ℹ️ О проекте

🎯 Цель проекта

Автоматический сбор и обновление IPTV плейлистов с различных сайтов-доноров. Все плейлисты обновляются каждые 6 часов через GitHub Actions и локальные слушатели.

🏗️ Архитектура

  • Pokaz.me - основная ветка main, сборка через Puppeteer
  • Vsetv.click - ветка feature/vsetv-click, сборка через Playerjs
  • Ontivi.net - ветка feature/ontivi, сборка с перехватом сетевых запросов

⚙️ Технологии

  • Node.js + Puppeteer для парсинга
  • GitHub Actions для автоматизации
  • PowerShell скрипты для управления
  • Система бэкапов перед каждой сборкой

📁 Структура проекта

m3u/
├── main (pokaz.me)
├── feature/vsetv-click (vsetv.click)
├── feature/ontivi (ontivi.net)
├── playlists/         # Готовые плейлисты
├── backups/           # Автоматические бэкапы
├── logs/              # Логи слушателей
└── src/               # Исходные коды
    ├── build_*.js     # Скрипты сборки
    └── listeners/     # Слушатели триггеров

🔧 Управление проектами

Для переключения между проектами используйте:

.\switch-project.ps1 -Target status     # Текущий статус
.\switch-project.ps1 -Target pokaz     # Переключиться на pokaz.me
.\switch-project.ps1 -Target vsetv     # Переключиться на vsetv.click
.\switch-project.ps1 -Target ontivi    # Переключиться на ontivi.net
